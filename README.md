## Usage/Examples

Créer la base de données :

Connecter vous à votre server MySQL
Vous trouverez le script de création à la racine du projet.

```SQL
source script.sql
```

Installer et demarrer l'application
```bash
npm install
npm start
```
Adresse de l'application en local :\
http://localhost:3000/
